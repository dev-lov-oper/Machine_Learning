


This section of the repository focuses on performing a comprehensive Exploratory Data Analysis (EDA) on the classic Titanic dataset before building Machine Learning models.

---

## 📊 Objectives
* **Data Cleaning:** Handle missing values (`Age`, `Cabin`, `Embarked`) and correct data types.
* **Visualization:** Explore feature distributions and correlations (e.g., how gender and class impacted survival rates).
* **Insights:** Identify key patterns to guide feature engineering for future ML models.
* * **Univariate Analysis:** Analyze individual features in isolation (e.g., survival rate distribution, age distribution using histograms, and passenger class counts).
* **Bivariate/Multivariate Analysis:** Explore relationships between multiple variables (e.g., survival rates cross-tabulated by gender and class, or checking correlations using a heatmap).

## 🛠️ Tech Stack
* Python (Pandas, NumPy, Matplotlib, Seaborn)
